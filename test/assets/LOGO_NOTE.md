# Logo Image Instructions

Please place the KURO Performance logo image in this directory as `logo.png`.

## Requirements

- **Filename**: `logo.png`
- **Recommended dimensions**: 400x400 pixels or higher
- **Format**: PNG with transparent background (preferred)
- **Alternative formats**: JPG, BMP are also supported

## Usage

The logo is used in:
1. Landing screen (left panel) - Large display
2. Upload screen (top-left corner) - Small icon
3. Results screen (top-left corner) - Small icon

## Image Provided

The image file `kuro_rebranding_icon_full_clr_online.png` is available in the project root.
You can copy it to this location using:

```bash
cp ../kuro_rebranding_icon_full_clr_online.png logo.png
```

Or if you have a different logo file:

```bash
cp /path/to/your/logo.png logo.png
```

## Fallback

If no logo is found, the application will display a placeholder circular logo with "KURO" text.
